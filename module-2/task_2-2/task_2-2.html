1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. [Constants should be used for whenever you have a value of which you will not change later, as the value of a cosntant is persistent throughout the code. If you have a value that will change, you should use a variable. Not all values can be constants, as some values need to be changed throughout the code.]

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. [Parentheses affect the order of operations, the same way one calculates mathematical equations. In maths, parentheses are calculated first, then multiplication and division, and finally addition and subtraction.]

  3. How do we limit floating-point numbers to a given number of decimals?

    4. [We can limit floating-point numbers using .toFixed(), of where the number of decimals is specified in the parentheses.]

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. [By using .toFixed() on both values, and for example setting "2" in between the parentheses, we can compare the two values to see if they are equal, as it rounds the values to the specified number of decimals. In this case, both values would be rounded to 12.20, and thus be equal.]

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. [They return a boolean value, which is either TRUE of FALSE. We can use an if-statement to check if they are TRUE or FALSE.]

  6. How does the Math.random() function work, and what can it be used for?

    7. [It generates a random floating point number betwween 0 and 1, and can be used to for example generate dice rolls]

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. [Modulus is what remains after dividing two numbers. For example, 5 % 2 = 1, as 2 goes into 5 two times, with 1 remaining. It is useful for example when you want your script to check if a number is even or odd, as even numbers have 0 remaining when divided by 2, while odd numbers have 1 remaining.]
